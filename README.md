# SNAKE-贪吃蛇 程序文档

## 1. 项目简介
这是一个单文件实现的网页贪吃蛇游戏，核心文件为 `贪吃蛇.html`。

主要特点：
- 支持手动与自动驾驶（AI）两种模式。
- 支持穿墙模式开关。
- 支持多档速度、棋盘尺寸、主题、语言切换。
- 支持本地排行榜（机器人/人工分组）。
- 支持设置与排行榜持久化（`localStorage`）。
- 自适应棋盘模式下支持侧边栏布局与悬停展开。

## 2. 运行方式
1. 直接用浏览器打开 `贪吃蛇.html`。
2. 无需安装依赖，无构建步骤。

## 3. 界面与功能
- 顶部状态区：显示系统状态、饥饿倒计时、当前分数。
- 游戏区：Canvas 绘制蛇、食物、网格和动画效果。
- 控制面板：
- 速度：`150/80/40/15/8`（慢速到极限）。
- 棋盘：`10x10` 到 `40x40`，以及 `auto` 自适应。
- 主题：10 套视觉主题。
- 语言：中文 / English。
- 功能按钮：自动驾驶开关、穿墙开关。
- 排行榜：机器人记录与人工记录分别展示，支持清空。

## 4. 操作说明
- 方向键：`↑ ↓ ← →` 控制方向（手动模式下有效）。
- `Space` / `Enter`：游戏结束后重新开始。
- `Z`：自动驾驶开关。
- `X`：穿墙模式开关。
- `C`：循环切换速度。
- `V`：循环切换主题。
- `B`：侧边栏开关（仅在 `auto` 自适应棋盘下有效）。

## 5. 游戏规则
- 吃到食物加分（每次 +10）。
- 蛇每移动一步，步数 `stepCount` +1。
- 非穿墙模式下撞墙死亡。
- 任意模式下撞到自身（按当前是否吃到食物区分尾部判定）会死亡。
- 启用自动驾驶时，死亡后会自动倒计时重开（默认 5 秒）。
- 存在“饥饿倒计时”：长时间未吃到食物会判定失败。

## 6. 排行榜规则
排行榜按模式分组：
- `bot`：自动驾驶记录。
- `human`：人工操作记录。

每条记录包含：
- 分数 `score`
- 时间 `time`
- 棋盘 `grid`（如 `16x16`）
- 对局时长 `duration`
- 步数 `steps`

排序逻辑：
1. 先按分数降序。
2. 若分数相同且棋盘相同，则按步数升序（步数更少排名更高）。
3. 仅保留前 `MAX_ENTRIES=10` 条。

## 7. 数据持久化
使用 `localStorage`：
- `cybersnake_settings`：保存速度、棋盘、自动驾驶、穿墙、语言、主题。
- `cybersnake_v1`：保存排行榜数据。

相关函数：
- `loadSettings()` / `saveSettings()`
- `loadScores()` / `saveScore()` / `renderLeaderboard()` / `clearLeaderboard()`

## 8. 核心代码结构
- 初始化与主循环：`init()`、`draw()`、`resetGame()`
- 设置更新：`updateSettings()`、`applyTheme()`、`applyLang()`
- 输入处理：`document.addEventListener("keydown", ...)`
- 游戏结束与重开：`gameOver()`、`startAutoRestartCountdown()`
- AI 决策：`autoPilot()` 及其路径规划/安全判定函数（BFS、哈密顿回路策略、模拟评估）
- 自适应布局：`applyAdaptiveLayout()`、`setAutoSidebar()`、`bindAutoSidebarHover()`

## 9. 可维护建议
- 若后续拆分文件，建议至少拆为：UI、游戏内核、AI、持久化四个模块。
- 为排行榜排序与碰撞判定补充单元测试，避免规则回归。
- 若要兼容移动端触控，可增加虚拟方向键与手势输入。
